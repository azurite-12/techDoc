Introducción

Python es un lenguaje de programación multi-paradigma de tipado fuerte y de alto nivel (alto nivel de abstracción, innecesario conocimiento de hardware). Es de los lenguajes de programación más populares para principiantes debido a la filosofía de simpleza de los desarrolladores de este (su nivel de simpleza solamente es rivalizado por Ruby).

Si bien Python no es conocido por su rendimiento, sigue siendo el lenguaje de programación predilecto a la hora de análisis estadístico, y también para desarrolladores independientes de novelas visuales. Python también es usado por aplicaciones muy populares, como YouTube y Spotify.

Preparaciones

Antes de empezar con Python en general, primero tenemos que instalar un par de cosas.

  • Si bien puedes escribir codigo incluso en el bloc de notas integrado de tu computador, es recomendable usar herramientas diseñadas para esto. Como:
    • Visual Studio Code (para ordenadores más potentes y modernos)
    • Notepad++ (para ordenadores más antiguos y/o de menor potencia).
  • El lenguaje mismo desde su página principal
  • Configurar tu ordenador para que éste muestre extensiones de archivos usando este tutorial.

Por último, crea un documento de texto en una localización de tu preferencia, y cambia la extensión de '.txt' a '.py'

Constantes y variables

En matemáticas, existen los conceptos de constantes y variables.

  • Constante: es un objeto matemático con un valor fijo e inmutable (Ejemplo: π y e).
  • Variable: es un objeto matemático cuyo valor varía y cambia dependiendo de sus circunstancias. Variables comunes son x, y o z. Estos objetos pueden representar un número, un vector, una matríz o una función.

Como Python es un lenguaje de tipado fuerte y dinámico, no hay forma de declarar constantes. Sin embargo, es buena práctica en todos los lenguajes de programación declarar tus constantes en mayúscula (ej: CONST=10) y tus variables en camelCase (ej: myVar=5).

Ahora practica tú en Python escribiendo lo siguiente:

            
              x = 4
              y = 6
              print(x+y)
              print(x+10)

Y ejecutelo. Sientete libre de cambiar los valores de las variables en el ejemplo anterior para experimentar.

Tipos de datos

Python posee 8 tipos de datos dentro de si mismo, pero los más básicos son los 4 siguientes:

  • str --> String, texto. Siempre va entre comillas dobles " o simples '.
  • Numéricos:
    • int --> Número entero.
    • float --> Número decimal.
  • Secuencias:
    • list --> Colección de valores ordenados, modificable.
    • tuple --> Colección de valores ordenados, inmutable.
    • set --> Colección de valores no ordenados, inmutable. No permite elementos duplicados dentro de éste.
  • bool --> Booleano. Solo toma dos valores: True y False.

Debido que python es de tipado dinámico, no es necesario especificar el tipo de dato antes de una variable a la hora de declararla. Python reconocerá que la variable myList en myList=[1,2,3] en tu código es una lista (list) automáticamente.

Operadores matemáticos

Python soporta múltiples operadores matemáticos para hacer cálculos y otros juegos con variables y constantes. Éstos son:

  • + | Adición (suma).
  • - | Sustracción (resta).
  • * | Multiplicación.
  • / | División.
  • % | Módulo (devuelve el resto que resulta de la división entre dos números).
  • ** | Potencia.
  • // | División de piso (devuelve la parte entera que resulta de la división entre dos números).

Sientete libre de copiar el ejemplo de la sección anterior y experimentar con los diferentes operadores.

Asignación de valores

En Python, los valores de que están al lado derecho del signo = se evaluan antes de asignarlos a la variable de la izquierda. Debido a esto, puedes usar variables anteriormente declaradas dentro de la misma linea de codigo. Por ejemplo: 

            
              x = 4
              x = x + 20
              print(x)
              
              out: 24

Como se muestra en este ejemplo el output es 24. Python ignora todo lo que hay a la izquierda del signo '=' hasta que termina de evaluar todo lo que hay a la derecha. Sientete libre de experimentar con el ejemplo anterior, añadiendo más operaciones y paréntesis. Ej: x = x * 3 * (x-4)

Operadores de asignación

Python tiene ciertos atajos para asignar valores a variables usando las mismas variables, algunos de estos son:

  • += | Ej: x+=3 --> x = x + 3
  • -= | Ej: x-=5 --> x = x - 5
  • *= | Ej: x*=2 --> x = x * 2
  • /= | Ej: x/=8 --> x = x / 8
  • **= | Ej: x**=3 --> x = x ** 3
Operadores de comparación

Estos operadores devuelven un valor True o False a la hora de evaluar los dos valores en los que se usan.

  • == --> Igual. Ej: si 3==3, el sistema devuelva True. Si 3==5 el sistema devuelve False.
  • != --> Desigual. Ej: si 3!=3, el sistema devuelva False. Si 3!=5 el sistema devuelve True.
  • > --> Mayor.
  • < --> Menor.
  • >= --> Mayor o igual.
  • <= --> Menor o igual.
Declaraciones condicionales

Existen una maneras para comparar valores y modificar el comportamiento del código:

  • if...else --> Declaras una condición con if y propones una condición. Si esta condición se cumple, señalas una instrucción a seguir para el script. Si no se cumple, la condición señalada anteriormente no se cumple y el codigo sigue ejecutandose despues de esto. Aunque puedes declarar else y señalar otra instrucción a ejecutarse. Ejemplo: 
                    
                  if num==3:
                    print("Tu número es 3")
                  else: 
                    print("Tu número no es 3")

    Está también la declaración elif, que te permite anidar más condiciones a un declaración if:

                    
                  if num > 3 :
                    print("Su número es mayor que 3")
                  elif num < 3:
                    print("Su número es menor que 3")
                  else:
                    print("Su número es 3")

    También existen operadores lógicos como and, or y not que evaluan múltiples comparaciones y devuelven True o False dependiendo del resultado. Ejemplo:

                    
                  if (num >= 5) and (num <= 10):
                    print("Su número está entre 5 y 10")
                  else:
                    print("Su número está fuera del rango 5-10")

Listas

Una lista es una estructura de datos básica. Es una colección de elementos ordenados y modificable. Se puede declarar una de éstas usando brackets [] y llenando éstos con valores separados por comas. Debido a su naturaleza, ésta está indexada de 0 hasta n-1, siendo n el número total de elementos de la lista. Esto quiere decir que es posible a acceder a cualquier de estos elementos, o un rango de ellos. Ejemplo: 

            
              miLista = [2,3,5,7] // Lista de n=4 elementos.
              print(miLista)
              print(miLista[0])
              print(miLista[2])
              print(miLista[3])
              print(miLista[4])

              out:
              [2,3,5,7]
              2
              5
              7
              IndexError: List index out of range.

Como se puede ver en el ejemplo, llamar a la variable retorna la lista completa. Si bien esto es útil para ciertos algoritmos, esto no sirve a la hora de realizar operaciones aritméticas. Si tratas de hacer miLista + 3 Python tirará un error. Para esto es necesario usar los índices correspondientes de cada elemento. Para esto es necesario escribir la variable de la lista seguido inmediatamente (sin espeacios) por el índice del valor al que quieres llamar. Como las listas son indexadas de 0 hasta n-1 (n= número total de valores de la lista), tienes que escribir miLista[3] para llamar al cuarto valor, miLista[5] para llamar al sexto valor, etc. Si llamas al valor n, sale un error, como señala en el ejemplo.

Un operador útil de listas son los dos puntos :. Estos simbolizan un rango y pueden ser usados para llamar a una sub-lista de la lista original. Ejemplo:

            
              miLista=[2,3,5,7,9]
              print(miLista[0:3])
              print(miLista[2:5])
              print(miLista[0:5])
              print(miLista[2:])
              print(miLista[:4])

              out:
              [2,3,5]
              [5,7,9]
              [2,3,5,7,9]
              [5,7,9]
              [2,3,5,7]

Tip:siendo k un número entero, usar k: sin valor final devuelve una sub-lista conteniendo todos los valores de k en adelente (de k hasta n-1). Usar :k devuelve una sublista de los primeros k valores de la lista.

    Funciones útiles de listas:
  • .append(elem) --> inserta el elemento elem al final de una lista. Ej: miLista.append(80) --> out: miLista = [2,3,5,7,9,80]
  • .insert(index,elem) --> Inserta el elemento elem en la posición index. Ej: miLista.insert(2,24) --> out: miLista = [2,3,24,5,7,9,80]
  • .extend(list) --> Inserta la lista provista como argumento al final de la lista a la que se le aplica esta función.
  • .pop(index) --> Elimina el elemento en el indice index de la lista a la que se le aplica esta función, dejar el argumento vacío (.pop()) elimina el último elemento de la lista.
  • .remove(elem) --> Elimina el específico elemento elem de la lista a la que se le aplica. Ej: miLista.remove(9) --> out: miLista = [2,3,24,5,7,80]
  • .clear() --> Elimina todos los elementos de una lista y devuelve una lista vacía [].
  • .count(elem) --> devuelve un int que corresponde al número total de instancias que el elemento elem está en la lista.
  • .reverse() --> Devuelve una lista con los elementos de la lista a la que se le aplica pero al revéz.
Loops (o ciclos)

En muchos lenguajes de programación, existen declaraciones que permiten ejecutar una (o varias) instrucciones de código múltiples veces, siempre que se les de una condición. Python no es diferente en este aspecto, y el lenguaje posee dos tipos de ciclos: while y for.

  1. while(condición) --> las instrucciones escritas dentro de este ciclo se ejecutarán repetidamente siempre y cuando la condición dentro de éste se cumpla. Hay que tener precaución y escribir instrucciones específicas para que el loop no se ejecute eternamente. 
                    
                  // 1
                  NUM = 3
                  while(NUM < 5):
                    print("Hola mundo")
                  
                  // 2
                  count = 0
                  while(count < 5):
                    print("Hola mundo")
                    count += 1

    El primer ejemplo repite la instrucción dentro del while (print("Hola mundo")) siempre que NUM sea menor a 5. Como NUM es una constante, el código se repetirá eternamente, y hay que forzarlo a detenerse con shift+F5.

    El segundo ejemplo, sin embargo, usa una variable, y dentro del while, esta variable se modifica con cada iteración. Lo que resultá en el while deteniendose una ves la variable count llega a 5. Osea, solo se imprime Hola mundo 5 veces en pantalla.

  2. for iterator in sequence --> este ciclo recibe dos valores. El primer valor va después del for y es la variable iteradora, usualmente denotada como i. Mientras que la secuencia puede ser cualquier rango de valores (usando la función range()). También cabe mencionar que la secuencia también recibe listas y strings. Contrario al while las instrucciones dentro de esta declaración se ejecutan una vez se cumple el rango de valores dado. 
                    
                  // 1 usando función range
                  for i in range(5):
                    print("Hola mundo " + str(i))

                  // 2 usando una lista
                  miLista = [2,3,5,7,9,11]
                  for i in miLista:
                    print("Hola mundo " + str(i))

    En el primer ejemplo, la instrucción dada dentro del for imprime Hola mundo 0 Hola mundo 1, etc. hasta que termina la secuencia range. Ésta función empieza de 0 y termina en n-1, siendo n el número entero dado a la función. Si a range(x,y) le das dos valores x e y, la secuencia empieza en x y termina en y-1. Ej: range(2,5) --> 2 3 4

    En el segundo ejemplo, la instrucción dada dentro del for imprime Hola mundo 2 Hola mundo 3 Hola mundo 5, etc hasta que termina de iterar todos los elementos de la lista. Puedes también poner una sublista usando el comando : en la secuancia. Ej: fpr i in miLista[:3] <-- Itera solo hasta el tercer elemento.

Funciones

En la programación, una función es un bloque de codigo que recibe un cierto numero de argumentos y que se puede llamar en otras partes del codigo para ejecutar una tarea. Son muy útiles a la hora de programar ya que evita la repetición de codigo, quedando así el script más limpio. Se declara con la declaración def.

            
              // Sintaxis
              def nombreFuncion(argumento0, argumento1, argumento2, ...):
                lorem ipsum
                lorem ipsum:
                  loremipsum

Para llamar a la función y que haga lo que se le escribió dentro, solo se escribe el nombre junto con los argumentos que se les quiere pasar. Ahora que sabemos como es la sintaxis de una función, veamos como definirlas y llamarlas con el siguiente ejemplos:

            
              // 1
              def miFuncion():
                print("Hola mundo")

              def miFuncion2(nombre):
                print("Mi nombre es " + nombre)

              miFuncion()
              miFuncion2("Diego")

              out:
              Hola mundo
              Mi nombre es Diego

              // 2
              def suma(x,y):
                print(x+y)
                return x + y

              def mayorQueTres(x):
                if x>3: print(str(x) + " es mayor que 3")
                else: print(str(x) + " es menor o igual que 3")

              suma(3,7)
              mayorQueTres(2)

              out:
              10
              2 es menor o igual que 3

En el ejemplo 2 vimos como las funciones son declaradas, definidas y llamadas. La declaración return dentro de una función, puesta al final de ésta, definirá qué elemento devolverá ésta una vez llamada. Más detalles de funciones aquí
